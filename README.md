# 🌍 EcoPulse AI

## Carbon Footprint Awareness Platform

EcoPulse AI helps users understand and reduce their environmental impact by calculating carbon emissions and providing practical suggestions.

---

## Features

✅ Carbon footprint calculator

✅ Eco score (0-100)

✅ Emission breakdown chart

✅ Personalized eco suggestions

✅ History tracking

---

## Tech Stack

- React
- JavaScript
- Vite
- Recharts

---

## Inputs

- Transportation
- Distance Travelled
- Electricity Usage
- Diet Type
- Waste Generated

---

## Output

- Total CO₂ Emission
- Eco Score
- Breakdown Visualization
- Suggestions
- Previous Calculations

---

## Installation

```bash
npm install
npm run dev
```
